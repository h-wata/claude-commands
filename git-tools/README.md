# git-tools

Git history investigation and change tracking commands.

## Installation

```
/install git-tools@h-wata/claude-commands
```

## Commands

### `/git-history`
Investigate git history and track changes.

```
/git-history src/main.py history
/git-history "handleRequest" search
/git-history src/main.py:42 blame
/git-history v1.0.0..v2.0.0 diff
```
